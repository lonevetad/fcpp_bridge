Created 5 new files implementing the OutputChannel hierarchy:

  ipc/output_channel.py
    ABC with send(name, payload), close() (no-op default), clone() (copy.copy).
    Prototype pattern: clone() returns a shallow copy; subclasses override for
    proper independence where needed.

  ipc/logging_output_channel.py
    LoggingOutputChannel(level, logger_name).
    send() emits [name] payload via bridge logger at the configured level.
    clone() reuses the same logger reference (loggers are singletons).

  ipc/file_output_channel.py
    FileOutputChannel(path_or_stream, format).
    format="json" (default) → {"name":…,"payload":…}\n per send.
    format="csv" → name,payload\n per send.
    When given a path: owns the stream, closes it in close().
    When given a stream: borrows it, clone() never owns.

  ipc/callback_output_channel.py
    CallbackOutputChannel(fn).
    send() calls fn(name, payload). close() is a no-op. clone() wraps same fn.

  ipc/proxy_output_channel.py
    ProxyOutputChannel(mode="sequential"|"parallel").
    add_channel(ch) → int  (channel ID for removal).
    remove_channel(id)     (no-op if missing).
    send() sequential: calls channels in registration order.
    send() parallel:   ThreadPoolExecutor dispatches concurrently.
    close() calls close() on all sub-channels.
    clone() deep-copies sub-channels via ch.clone().

Updated files:

  ipc/__init__.py
    Added imports + __all__ entries for all 5 new classes.

  ipc/device_manager.py
    __init__: added output_channel: Optional[OutputChannel] = None kwarg.
    self._output = output_channel or LoggingOutputChannel() (default).
    start_all(): failure → also calls self._output.send("start_all", {...}).
    connect_all(): failure → also calls self._output.send("connect_all", {...}).
    step_all(): failure → also calls self._output.send("step_all", {...}).
    close_all(): failure → also calls self._output.send("close_all", {...}).

  TESTS (+15 in tests/ipc/test_output_channel.py):
