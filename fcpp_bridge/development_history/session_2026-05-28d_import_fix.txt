  PYTHONPATH=.. /usr/bin/python3 examples/worker_role_assignment.py

Error:
  ModuleNotFoundError: No module named 'examples'

Root cause:
  All 6 example files contained:
    from examples._example_utils import report_validation, report_transpilation

  This bare `examples` import requires src/fcpp_bridge/ to be on sys.path so
  Python can find `examples` as a top-level package.  Neither invocation style
  puts src/fcpp_bridge/ on sys.path:
    - PYTHONPATH=..  (from src/fcpp_bridge/) → sys.path gets src/
    - PYTHONPATH=src (from repo root)        → sys.path gets src/
  In both cases Python sees fcpp_bridge/ under src/, but not examples/ directly.

Fix:
  Changed the import in all 6 example files:
    from examples._example_utils import ...
  →
    from fcpp_bridge.examples._example_utils import ...

=== KEY DESIGN DECISIONS ===

1. Fully-qualified import path (not relative import)
   Relative imports (from ._example_utils import ...) only work when the file
   is executed as a module (python -m fcpp_bridge.examples.worker_role_assignment).
   Since the examples are run as scripts (__name__ == '__main__'), relative imports
   raise ImportError. The fully-qualified path from fcpp_bridge.examples._example_utils
   works in both script mode and module mode.

2. sys.path layout under PYTHONPATH=..
   When running from src/fcpp_bridge/:
     sys.path[0] = src/fcpp_bridge/examples/  (the script's directory)
     sys.path also includes src/  (from PYTHONPATH=..)
   fcpp_bridge is found at src/fcpp_bridge/, so fcpp_bridge.examples resolves
   correctly. The bare name 'examples' was never findable.

3. No __init__.py trick possible
   An __init__.py in examples/ cannot fix the sys.path problem: it only marks
   the directory as a package but does not add its parent to sys.path. The
   parent (src/fcpp_bridge/) must be explicitly on the path for 'examples' to
   be a top-level importable name — which would require PYTHONPATH=../..
   from src/fcpp_bridge/, making the two-entry path requirement (src/ for
   fcpp_bridge AND src/fcpp_bridge/ for examples) ugly and fragile.
