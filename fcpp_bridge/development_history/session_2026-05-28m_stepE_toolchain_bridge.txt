  E.1 — SwarmProcess.latest_snapshot()

    Added to swarm_process.py:
      _latest_snapshot: Optional[SwarmSnapshot] = None  (in __init__)

      def latest_snapshot(self) -> Optional[SwarmSnapshot]:
          """Return the snapshot from the most recent update, or None."""
          return self._latest_snapshot

      def _dispatch_update(self, snapshot: SwarmSnapshot) -> None:
          self._latest_snapshot = snapshot
          super()._dispatch_update(snapshot)

    Intercepts every push update before forwarding to the listener
    pipeline, so latest_snapshot() always reflects the most recent
    state without requiring a separate pull call.

  E.2 — abstract_example.py: full rewrite

    REMOVED abstract methods:
      - round_step(round_num, positions, states) → (positions, states)
      - initial_states(positions) → dict

    ADDED abstract property:
      - aggregate_class → Type  (the @aggregate_function class)

    ADDED optional properties (with defaults):
      - build_dir → Path  (examples/.fcpp_build/)
      - cpp_dir   → Path  (examples/.fcpp_cpp/)

    UPDATED hook signature:
      - on_round_complete(round_num, snapshot: Optional[SwarmSnapshot])
        was: on_round_complete(round_num, positions, states)

    REWRITTEN run(num_rounds):
      1. AggregateValidator.validate(self.aggregate_class)
      2. Transpiler(self.aggregate_class).generate() → cpp_code
      3. Compiler(cache_dir=build_dir, cpp_dir=cpp_dir)
            .get_or_compile(cpp_code, self.log_prefix) → binary
      4. SwarmProcess(binary_path=binary, num_nodes=len(positions))
      5. swarm.add_listener(_on_snapshot)  [before start()]
      6. swarm.start()
      7. swarm.add_node_explicit(nid, pos)  [for each initial_positions() entry]
      8. on_simulation_start()
      9. for round_num in range(num_rounds):
             swarm.step()
             on_round_complete(round_num, swarm.latest_snapshot())
      10. close all open log_files
      11. swarm.close()
      12. on_simulation_end()

    _on_snapshot listener:
      - Opens log file + writes header on new node_id
      - Appends log_line per active node
      - Closes log file for nodes absent from snapshot.nodes
